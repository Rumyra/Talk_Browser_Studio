Wisdom of Crowds

But it also talked about the the wisdom of crowds. That is ask enough people and you will average the correct answer.

First observed in Plymouth in 1906, where a crowd was asked to estimate the weight of an ox, although all the individual guesses where wrong, the average of those guesses was 1207 lbs, the ox weighed 1198 lbs - within 1% of the correct answer

Fun fact, this doesn't just happen with humans. The golden shiner fish is a fish that prefers shade, but is much better at finding it in schools of other golden shiners, than on it's lonesome.

function drawCard() {
	const card = new Path2D( ...path data here... )
	ctx.fillStyle = colour;
	ctx.fill(card);
}

The canvas has a path2d constructor. Which you can use to _save_ a path to fill or stroke later. So you can use normal canvas 2d methods, or you can dump in an svg path attribute into this bad boy.
